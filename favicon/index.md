### Import:
```html
<script
	type="module"
	src="https://rm-netsu.github.io/yaclib-barebones/favicon/index.js"
></script>
```

### Examples:
```html
<yac-favicon
	url="https://google.com"
	size="16"
></yac-favicon>
```


### Attributes:
- **url**: site URL.
- **size**: size of favicon. *Default: 64*
